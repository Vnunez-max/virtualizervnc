# Modular Synergy Map

The project is not a flat list of scripts. It is a cumulative geometric evidence system.

```text
V3.4.2 -> C1.x -> U1.x -> L1.x -> G1.x -> D1.x -> X2.0 -> X3.0
```

## Layers

### V3.4.2

Residual evidence stratifier. Frozen upstream. Must not be modified.

### C1.x

Residual evidence geometry hypothesis validators.

- C1.0: individual residual evidence hypothesis validation
- C1.1: collective residual evidence hypothesis validation

### U1.x

Geometric purity gates.

- U1.0: clean grid geometry purity gate
- U1.1: subobject clean grid geometry purity gate
- U1.1-CAL: calibration/hysteresis support

### L1.x

Observed support domain stratification and deferred resolution.

- L1.0: observed support domain stratifier
- L1.1: observed support domain calibration layer
- L1.2: deferred domain subsupport resolver
- L1.2-CAL: deferred line-like fragment calibrator

### G1.x

Deferred family association and trainable calibration.

- G1.0: deferred line family resolver
- G1.0-CAL V1: trainable deferred family calibrator

### D1.x

Complementary deferred observed-line analysis.

- D1.0: simple lineality auditor inside deferred
- D1.1: deferred linear role classifier

### X2.0

Real single-script fusion layer. It consumes upstream maps and fuses evidence into line-study support and future-pool support.

### X3.0

Trainable-aware complete Geometric Evidence Unit runtime.

- consumes UNIT, G1.0-CAL V1, D1.0 and D1.1 outputs
- verifies readable trainable model assets
- writes trainable influence maps
- preserves source-bit traceability
- remains fusion/orchestrator, not a trainable monolith

## Core Principle

Each layer may add interpretation only when it preserves or improves traceability.

```text
more interpretation + less traceability = reject
more interpretation + equal/better traceability = candidate
```
